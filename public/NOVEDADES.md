### 🗓️ 21 de Abril de 2026

### 🗓️ 20 de Abril de 2026

**🎉 Productos**: Ahora puedes agregar imágenes a los productos desde el formulario de edición. Las imágenes se muestran en la tabla de productos y se almacenan en la nube con CDN global.

**✨ Productos**: Las imágenes se redimensionan y comprimen automáticamente para optimizar el almacenamiento y la carga.

**🔍 Búsqueda de Productos**: Ahora puedes usar búsqueda avanzada. Escribe `led+cronos` para encontrar productos que contengan ambas palabras, o `filtro aire` para buscar cualquiera de las dos. También puedes usar comillas para búsqueda exacta como `"LED-123"`.

**✨ Ventas Rápidas**: El botón de pago ahora dice "Registrar pago" para mayor claridad.
